# Audio Files

## celebration.mp3
This file should contain a fun celebration sound that plays when the fun modal appears.

You can add any celebration sound effect here, such as:
- Party horn sound
- Applause
- Cheering crowd
- Confetti popping sound
- Fun music clip

The audio file should be named `celebration.mp3` and placed in this directory.

**Note:** The audio is set to 50% volume and will gracefully handle cases where autoplay is blocked by the browser.
